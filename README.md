# 快速下载器  FastDownloader

因为 IDM 软件试用到期了，所以自己写一个来用。

目前主要功能已实现，可下载各类大小文件

## 目前实现的功能：
1. 多线程同时下载一个文件
2. 多线程下载进度展示
3. 跟迅雷类似的监听复制到内存中的下载地址 自动弹下载框
4. 关闭程序默认最小化到托盘，可以在托盘中彻底退出程序
5. 本地化，目前只支持中文和英文 非中文系统语言都显示成英文
